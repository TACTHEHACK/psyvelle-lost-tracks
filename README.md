# Psyvelle Lost Tracks

*Searching for the missing Psyvelle releases formerly available in 2026.*

This project is looking for listeners or collectors who preserved recordings released under the artist name **Psyvelle** before the catalogue disappeared from Spotify, Apple Music and other services.

## Current status

- **21 albums / 204 tracks identified**
- **122 tracks currently available in the private collection**
- **12 complete albums marked OK**
- **1 incomplete album**
- **8 albums with no local tracks**

`✅ OK` means the album is present in the private collection. A fraction such as `✅ 5/10 OK` means only that number of tracks is present.

## Can you help?

Please comment in [Issue #1](https://github.com/TACTHEHACK/psyvelle-lost-tracks/issues/1) if you remember Psyvelle, preserved any recordings, or know more about PV Studio, Anthony Coleman or the distributor.

Please do not post public download links to copyrighted recordings. Material can first be verified through metadata, durations, hashes and audio fingerprints.
